Personal Life

Site Development

This site was developed in Codio using numerous developmental tools including: Node.js, socket.io and Nodemon. Bootstrap elements were added and customised with CSS to provide responsiveness to the across different media platforms. The design was extensively thought about before CSS was implented, to create a smooth, minimalistic and muted feel. Demo code was provided and external resources were used (referenced) to understand and implement different JavaScript commands for functionality of the chat page.
